# Calculadora Web

Esse projeto é uma calculadora simples feita com **HTML, CSS e JavaScript**.  
O objetivo foi treinar lógica de programação, JavaScript básico e a separação dos arquivos (HTML, CSS e JS).

---

## O que essa calculadora faz

- Soma, subtração, multiplicação e divisão  
- Porcentagem (%)  
- Inverter sinal (+/-)  
- Botão AC para limpar tudo  
- Mostra erro quando a conta é inválida  

---

## Tecnologias usadas

- HTML  
- CSS  
- JavaScript puro (sem framework)

---

## Organização do projeto

calculadora
├── index.html
├── style.css
├── script.js
└── README.md


---

## Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador  
3. A calculadora já vai funcionar  

---

## Por que fiz esse projeto

Fiz esse projeto para praticar JavaScript, HTML, CSS e JavaScript em arquivos diferentes.
